K intelligence
Suite 구성요소

신뢰도 높은 검증 데이터에 기반한 한국형 AI

  • Infra
  • RAI
  • Studio
  • Model
  • RAG
  • Agent

  • Infra

    AI 특화 고객 맞춤형
    보안 강화 클라우드 플랫폼

    한국에 위치한 데이터 센터를 통해 데이터 주권 보호
    고객 소유 전용키로 데이터 이중 보호, 국내 법률/규제 준수

  • RAI

    책임있는 AI를 실현하기 위한
    RAI 프레임워크

    Responsible AI Center(RAIC), AI 윤리원칙 (ASTRI), AI 원칙, 정책,
    기술을 수용한 개발 프로세스 및 RAI 문화 조성

  • Studio

    데이터 수집부터 AI 학습
    배포, 서빙 및 모니터링까지
    통합 제공하는 플랫폼

    SaaS형 Studio, On-premise(구축형) Studio 제공

  • Model

    K-Data를 학습하여
    한국의 정체성을 담은 모델

    KT자체모델, 글로벌 협력모델, 오픈 소스 기반 모델

  • RAG

    다양한 타입의 비정형 문서를 AI가
    이해할 수 있는 형태로 변환하는
    데이터 처리 및 검색 기술

    Search AI, Document AI 제공

  • Agent

    고객의 상황을 이해하고
    판단하여 스스로 결정하는
    에이전트

    고객 맞춤형 에이전트, KT 서비스 에이전트 제공

  • AI 특화 고객 맞춤형 보안 강화 클라우드 플랫폼

    한국에 위치한 데이터 센터를 통해 데이터 주권 보호 고객
    소유 전용키로 데이터 이중 보호, 국내 법률/규제 준수

  • 책임있는 AI를 실현하기 위한 RAI 프레임워크

    Responsible AI Center(RAIC), AI 윤리원칙 (ASTRI), AI 원칙, 정책, 기술을 수용한 개발 프로세스 및 RAI 문화 조성

  • 데이터 수집부터 AI 학습 배포, 서빙 및 모니터링까지 통합 제공하는 플랫폼

    SaaS형 Studio, On-premise(구축형) Studio 제공

  • K-Data를 학습하여 한국의 정체성을 담은 모델

    KT자체모델, 글로벌 협력모델, 오픈 소스 기반 모델

  • 다양한 타입의 비정형 문서를 AI가 이해할 수 있는 형태로 변환하는 데이터 처리 및 검색 기술

    Search AI, Document AI 제공

  • 고객의 상황을 이해하고 판단하여 스스로 결정하는 에이전트

    고객 맞춤형 에이전트, KT 서비스 에이전트 제공
