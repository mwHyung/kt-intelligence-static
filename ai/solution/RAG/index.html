RAG

KT의 RAG(Retrieval-Augmented
Generation) 기술은 AI 에이전트가 사용자의
질문에 정확하고 개인화된
답변을 제공하도록 지원
합니다. 이를 통해 고객은 더 빠르게, 더 믿을 수 있는
정보를 실시간으로 얻을 수 있습니다.

Customer Value

RAG 기술이 바꾸는
AI 응답의 기준

KT의 RAG 기술은 외부 데이터와 내부 지식을 실시간으로 참조해, 신뢰할 수 있는 답변을 제공합니다.

  • 정확하고 신뢰할 수 있는
    정보 제공

    외부와 내부 정보를 실시간으로 참조해, 최신성과 신뢰성을 갖춘 답변을 제공합니다.

    이를 통해 언제 어디서나 정확한 정보와 맞춤형 솔루션을 전달합니다.

  • 업무 생산성 및 퀄리티 향상

    KT의 RAG 기술은 정보 탐색과 검증을 자동화해 정확도를 높이고, 내부지식 검색,
    보고서 작성까지 다양한
    업무의 효율과 만족도를 향상시킵니다.

RAG 특화 모델 3종

  • Embedding Model

    • What is

      단어, 문장, 문서 등을 벡터 형태로 변환

      한・영 검색에 특화된 KT 자체 개발 임베딩 모델

      한국어・영어에 최적화된 언어 처리 기술

      (금융, 공공, 의료, 법률, 커머스 등 다양한 산업 분야에서 고르게
      검증된 성능)

      글로벌 수준의 성능을 갖춘 KT 독자 모델

      (MTEB 성능 검증)

    • Benchmarks

      한국어 데이터 전반적 성능 우위

      (vs. bge-m3)

      금융, 공공, 법률, 의료, 커머스 분야

      영어 데이터 전반적 성능 우위

      (vs. bge-m3)

      MTEB(eng, v2) 기준 평균 11%p 우위

    • What is

      한/영 검색에 특화된 KT 자체 개발 Reranking
      모델

    • Advantages

      한국어・영어에 최적화된 언어 처리 기술

      (금융, 공공, 의료, 법률, 커머스 등 다양한 산업 분야에서 고르게
      검증된 성능)

    • What is

      문서에 부가 정보를 더하고, 검색 방식 최적화와 응답 정합성 검증까지 지원하는 AI 모델

      Modular & Agentic RAG를 위한 특화된 Instruction 모델

      Llama K/믿:음 기반으로 RAG 특화 Instruction을 SFT하여 RAG 성능 향상

      RAG 성능 향상을 위한 Data Enrichment, Query Reasoning, Verification 기능 제공

      ChatGPT-4o와 같은 대형 LLM 모델과 유사 또는 좋은 성능을 내는 RAG Task 기능 제공

    • Benchmarks

      Data Enrichment, Query Reasoning, Verification 관련 성능 분석

Solution 02

Document AI

텍스트, 표, 이미지 등 문서 요소를 자동 인식해
AI가 이해하기 쉬운 구조로
정리하는 분석
솔루션입니다. 문단 순서, 표 구조, 문서 유형까지
정밀하게 처리해 RAG 성능을 높입니다.

Document AI 특징

  • 최적화된 전처리 엔진

    • 문서를 구조분석하여 AI가 이해하기 쉬운
      형태로 전처리하는 엔진

  • 다양한 타입의 문서 양식 지원

    • HWP/PDF/Word/Excel/PPT/Txt/JPG 등 문서 타입 대부분 지원

  • 문서 내 요소 추출 기능

    • 복잡한 표 구조인식, 문단 레이아웃 형태
      추출

  • 문서 처리 특화 기능

    • 단락 간 포함관계 이해, 문서 유형 자동분류,
      키워드 추출

주요 기능

  • OCR

    • 문서 내에 있는 텍스트를 추출하여 디지털
      텍스트로 변환하는 기술
    • 인쇄본/디자인 서체에 강인한 문자 인식
      모델

  • Table Detection / Recognition

    • 문서 내에 있는 표에 대해 구조를 분석하고
      이를 데이터 형태로 변환하는 기술
    • 복잡한 형태의 표 구조 인식, 타사 대비 높은 정확도

  • Layout Analysis

    • 문서 내에서 텍스트 / 제목 / 표 / 그림 등
      다양한 요소들을 구분하고, 페이지 내의
      단락 및 문장의 구조를 분석하는 기술

  • Chart

    • 문서 이미지 내 차트 이미지를 인식하고 이를
      표 형태로 변환하는 기술
    • Bar / Line / Pie / 멀티 스택형 차트 데이터

  • Diagram Recognition

    • 수식/순서도 다이어그램 인식 기술

  • D-Parser

    • 문서 파일 확장자 별로 CPU 전용 고속 추출 분기 처리
