on RAG

KT의 RAG(Retrieval-Augmented Generation) 기술은 AI 에이전트가 사용자의 질문에 정확하고 개인화된
답변을 제공하도록 지원합니다. 이를 통해 고객은 더 빠르게, 더 믿을 수 있는 정보를 실시간으로 얻을 수 있습니다.

Customer Value

RAG 기술이 바꾸는 AI 응답의 기준

KT의 RAG 기술은 외부 데이터와 내부 지식을 실시간으로 참조해, 최신 정보 근거 기반의 답변을 제공합니다.
반복적인 정보 탐색과 검증을 줄여 정확도를 높이고, 언제 어디서나 신뢰할 수 있는 정보와 맞춤형 솔루션을 제공합니다.

  • 시스템 강건성

    외부 공격 등의 리스크에도 AI 시스템이 안정적으로 작동할 수 있도록 설계합니다.

K on RAI Tool

KT는 안전하고 책임 있는 AI를 위해, 전 생애주기에 Responsible AI(RAI) 원칙을 적용합니다.
자체 개발한 AI 모델과 에이전트 출시 전, RAI 기준에 따라 위험 요소를 사전에 식별하고 체계적으로 평가·완화합니다.

곧 업데이트될 예정입니다.

  • 01

    Data Cleansing Tool

    학습 전 단계에서 민감 정보와 유해 콘텐츠를 제거합니다.
    AI 모델이 불필요하거나 위험한 데이터를 학습하지 않도록 필터링하여, 학습 품질과 윤리적 안정성을 확보합니다.
    • PII Filter
    • Toxic Filter
  • 02

    Model Reliability

    모델이 생성하는 응답을 사전에 평가합니다.
    AI가 유해한 응답을 생성하지 않도록 테스트하고, 반복 점검을 통해 강건성과 안전성을 지속적으로 강화합니다.
    • Safety Evaluation Solution
    • Red Teaming Solution
  • 03

    RAI Guardrail Tool

    운영 중에도 입력과 출력을 실시간으로 점검합니다.
    부적절하거나 민감한 콘텐츠를 필터링하며, Microsoft와 공동 개발한 한국어 특화 필터도 포함되어 있습니다.
    • Content Safety Filter (CSF)
    • KT Guardrail (공개 예정)
  • 04

    RAI Monitoring Tool

    운영 중 발생할 수 있는 리스크를 실시간으로 감지하고 분석합니다.
    AI 서비스가 사회적 이슈를 유발하지 않도록 지속적인 모니터링 체계를 제공합니다.

KT-MS CSF
전략 협업

KT는 AI가 한국어를 제대로 이해하고, 불쾌하거나 유해한 콘텐츠를
걸러낼 수 있도록 Microsoft와 함께 한국어에 특화된 'Content Safety
Filter(CSF)'를 만들었습니다.
해외로 데이터가 넘어가지 않도록 설계해, 더 안전하게 지킬 수 있습니다.
