Key Feature

RAG 기술이 바꾸는 AI 응답의 기준

KT의 RAG 기술은 외부 데이터와 내부 지식을 실시간으로 참조해, 최신 정보와 근거 기반의 답변을 제공합니다.
반복적인 정보 탐색과 검증을 줄여 정확도를 높이고, 언제 어디서나 신뢰할 수 있는 정보와 맞춤형 솔루션을 제공합니다.

  • Planning / Reasoning

    복잡한 문제를 스스로 분석하고, 해결 가능한 하위 작업(Sub-task)으로
    나눈 뒤 계획과 추론을 통해 처리합니다.

  • Memory

    사용자 정보와 과거 대화 이력을 기억해, 일관성 있고
    개인화된 응답을 제공합니다.

  • Tool

    검색, API, 데이터베이스 등 외부 도구와
    연동해 필요한 작업을 수행합니다.

  • Action

    외부 시스템과 상호작용하며 행동을 실행하고,
    결과를 반영해 최종 응답을 제공합니다.

Agent Tech

에이전트를 빠르고 정확하게 개발하고, 상황에 따라 최적의 AI 모델과 도구를 자동으로 연결해 실행합니다.

템플릿을 표준화한 Agent Pattern을 제공합니다.

Solution 02

Multi-Model Orchestrator

AI 응답 품질과 비용 효율을 동시에 잡는
스마트 라우팅 기술

Orchestrator는 질문의 성격에 따라 가장 적합한 AI 모델과 도구를 자동으로 연결해주는 스마트 라우팅 기술입니다
복잡한 요청에는 문서 검색(RAG), 계산기 등 보조 도구도 함께 활용해 더 정확하고 빠른 응답을 제공하며,
불필요한 고비용 모델 사용은 줄여 효율적인 AI 운영이 가능합니다.

Multi-Model Orchestrator 란?

사용자의 요청에 따라 적절한 모델을 알아서 선택,
연결해주는 Model 라우팅 기술입니다.

Orchestrator 특징

  • 실시간 모델 라우팅
    (Real-time Dynamic Routing)

    사용자의 쿼리를 분석해 가장 적합한 AI 모델을 실시간으로 예측·선택하고,
    즉시 연결합니다.

  • 비용 최적화
    (Cost Optimization)

    유사한 응답 품질을 제공할 수 있는 모델이 있다면, 불필요하게 고비용의 기본 모델을 사용하지 않고
    더 효율적인 모델로 자동 라우팅합니다.

  • 무중단 통합
    (Seamless Integration)

    Orchestrator를 사용하는 애플리케이션은 어떤 모델이 사용되는지 신경 쓸
    필요가 없습니다.
    모델이 추가되거나 제거되어도 별도 수정 없이 그대로 동작합니다.

  • OpenAI 호환 지원
    (Supports OpenAI Spec)

    OpenAI API 규격을 준수해, 별도의 추가 작업 없이 엔드포인트 변경만으로 쉽게
    적용할 수 있습니다.
