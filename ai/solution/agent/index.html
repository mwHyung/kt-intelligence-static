on Agent

주어진 목표를 달성하기 위해 환경을 인식하고, 작업을 계획하며 도구를 활용해 자율적으로 행동하는
고객 맞춤형 AI 에이전트입니다. 이를 통해 반복적인 업무를 줄이고, 더 빠르고 정확한 의사결정을 지원합니다.

Key Feature

RAG 기술이 바꾸는 AI 응답의 기준

KT의 RAG 기술은 외부 데이터와 내부 지식을 실시간으로 참조해, 최신 정보와 근거 기반의 답변을 제공합니다.
반복적인 정보 탐색과 검증을 줄여 정확도를 높이고, 언제 어디서나 신뢰할 수 있는 정보와 맞춤형 솔루션을 제공합니다.

  • Planning / Reasoning

    복잡한 문제를 스스로 분석하고, 해결 가능한 하위 작업(Sub-task)으로
    나눈 뒤 계획과 추론을 통해 처리합니다.

  • Memory

    사용자 정보와 과거 대화 이력을 기억해, 일관성 있고
    개인화된 응답을 제공합니다.

  • Tool

    검색, API, 데이터베이스 등 외부 도구와
    연동해 필요한 작업을 수행합니다.

  • Action

    외부 시스템과 상호작용하며 행동을 실행하고,
    결과를 반영해 최종 응답을 제공합니다.

K on Agent 핵심 기술

Agent Tech

Agent Pattern과 Multi-Model Orchestrator를 기반으로,
에이전트를 빠르고 정확하게 개발하고, 상황에 따라 최적의 AI 모델과 도구를 자동으로 연결해 실행합니다.

Agent Pattern

Agent를 정확하고 빠르게 개발할 수 있도록, 목적별 아키텍처와
템플릿을 표준화한 Agent Pattern을 제공합니다.

Multi-Model Orchestrator

사용 목적에 따라 가장 적합한 AI 모델과 도구를 자동으로 선택해주는
기능으로, 복잡한 설정 없이 최적의 성능을 구현할 수 있도록 지원합니다.

Solution 01

Agent Pattern

Agent 개발을 위한 표준 프레임워크

K on Agent는 목적과 상황에 맞게 표준화된 Agent Pattern을 기반으로, Agent를 효율적으로 개발할 수 있도록 합니다.
이를 통해 고객은 개발에 드는 시간과 비용을 줄이고 보다 안정적이고 일관된 품질의 Agent 서비스를 빠르게 확보할 수 있습니다.

영상영역

Agent Pattern 이란?

AI Agent의 빠른 개발과 배포를 위해 필요한 템플릿, 도구, 아키텍처를 사전에 구성한 프레임워크입니다.

Agent Pattern 특징

  • 01

    신속한 개발 환경 제공

    사용자는 목적에 맞게 설계된 Well-Architected Agent 템플릿을 활용하여 개발 시간을 대폭 단축할 수 있습니다.
    다양한 도메인과 상황에 맞춘 패턴을 재사용함으로써 개발 생산성이 향상됩니다.
    • KT Genie TV Media Agent 기준, 개발 속도 OO배 향상)
  • 02

    신속한 개발 환경 제공

    사용자는 인프라(IaC), 코드(UI 컴포넌트, API, 공통 로직), 프레임워크 등 Agent 개발에 필요한 모든 요소를
    표준화된 형태로 제공받을 수 있습니다. 이를 통해 반복적인 작업은 줄이고, 핵심 기능 개발에 집중할 수 있습니다.
  • 03

    개발 퀄리티 보장

    표준화된 Pattern 기반 개발 방식을 통해, 누구나 일정 수준 이상의 품질을 갖춘 Agent를 안정적으로 구현할 수 있습니다.
    또한 프롬프트 관리, 로깅, 모니터링, 성능 평가 등 전 과정에 대한 명확한 가이드라인을 제공해, 개발 효율성과 안정성을
    함께 높여줍니다.

Agent Pattern Framework

AI Agent의 빠른 개발과 배포를 위해 필요한 템플릿, 도구, 아키텍처를 사전에 구성한 프레임워크입니다.

  • Agent Pattern Layer

    Agent의 목적과 시나리오(미디어 추천, 고객 응대,
    문서 검색)에 따라 필요한 행동 양식을 구조화 한
    설계 단위

  • Pattern Type Layer

    Primitive Pattern : 단일 기능, 단일 목적을 가진
    기본 구조의 Pattern (정보 검색, 챗봇, 요약 등)

    Advanced Pattern : 복수의 Primitive Pattern을
    연결하거나, 계획, 실행, 검증과 같은 복합 구조를
    갖는 Pattern

  • Agent Capability

    Agent의 각 패턴이 제대로 동작하기 위한 기술 셋,
    인프라, 컴포넌트 계층

Solution 02

Multi-Model Orchestrator

AI 응답 품질과 비용 효율을 동시에 잡는 스마트 라우팅 기술

Orchestrator는 질문의 성격에 따라 가장 적합한 AI 모델과 도구를 자동으로 연결해주는 스마트 라우팅 기술입니다
복잡한 요청에는 문서 검색(RAG), 계산기 등 보조 도구도 함께 활용해 더 정확하고 빠른 응답을 제공하며,
불필요한 고비용 모델 사용은 줄여 효율적인 AI 운영이 가능합니다.

Agent Pattern 이란?

AI Agent의 빠른 개발과 배포를 위해 필요한 템플릿, 도구, 아키텍처를 사전에 구성한 프레임워크입니다.

Orchestrator 특징

  • 01

    실시간 모델 라우팅
    (Real-time Dynamic Routing)

    사용자의 쿼리를 분석해 가장 적합한 AI 모델을 실시간으로 예측·선택하고, 즉시 연결합니다.
  • 02

    비용 최적화
    (Cost Optimization)

    유사한 응답 품질을 제공할 수 있는 모델이 있다면, 불필요하게 고비용의 기본 모델을 사용하지 않고
    더 효율적인 모델로 자동 라우팅합니다.
  • 03

    무중단 통합
    (Seamless Integration)

    Orchestrator를 사용하는 애플리케이션은 어떤 모델이 사용되는지 신경 쓸 필요가 없습니다.모델이 추가되거나 제거되어도 별도 수정 없이 그대로 동작합니다.
  • 04

    OpenAI 호환 지원
    (Supports OpenAI Spec)

    OpenAI API 규격을 준수해, 별도의 추가 작업 없이 엔드포인트 변경만으로 쉽게 적용할 수 있습니다.
