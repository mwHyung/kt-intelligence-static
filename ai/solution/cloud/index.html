Infra

데이터는 국내에, 성능은 글로벌 그대로. 보안과 확장성을 갖춘 클라우드로
기업이 규제 부담 없이 클라우드를 자유롭고 안전하게 활용할 수 있도록 지원합니다.

Customer Value

데이터 주권 시대의
새로운 기준

KT는 MS와 협력하여, 강화된 보안으로 데이터를 안전하게 지키고
데이터 주권을 보호하는 한국적 Secure Public Cloud 서비스를 제공합니다.

Key Feature

K Infra의 기술적 차별점

데이터 주권, 보안 컴플라이언스, 암호화 보호, 자산 통제까지 모두 갖춰
기업이 안심하고 사용할 수 있는 신뢰형 클라우드 환경을 제공합니다.

  • 해당 국가 내 데이터 상주

    데이터는 국내에 저장하고 관리되어 국외 이전 없이 안전하게 보호됩니다.

  • 국내 보안 컴플라이언스 적용

    국내 기업에 적용되는 보안 법규를 철저히 준수하여 신뢰할 수 있는 클라우드 환경을 제공합니다.

  • 데이터 전 생애주기 보호

    데이터 저장부터 활용까지 데이터 수명 주기 전 단계에 걸쳐 강력한 암호화로 보호합니다.

  • 고객 자원 소유권 강화

    클라우드 운영자의 접근을 통제함으로써, 데이터에 대한 고객의 실질적 소유권을 강화합니다.
